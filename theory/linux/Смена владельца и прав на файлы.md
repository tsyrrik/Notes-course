
Базовые команды
- Посмотреть владельца и права: `ls -l`, `ll`
- Сменить владельца/группу: `sudo chown user:group path`
- Рекурсивно: `sudo chown -R user:group path/`
- Права: `sudo chmod 755 dir`, `sudo chmod 644 file`

Типичные паттерны на проде
- Код принадлежит пользователю деплоя, запись — только где нужно приложению:
  - `sudo chown -R deploy:deploy /var/www/app`
  - Папки для записи приложением (пример):
    - Laravel: `storage/`, `bootstrap/cache/`
    - Symfony: `var/`
  - Разрешить запись www-data только здесь:
    - Простой путь: `sudo chown -R www-data:www-data storage bootstrap/cache`
