
Интерактивный rebase (`-i`) позволяет:
- переупорядочить коммиты;
- объединить (squash);
- удалить ненужные;
- переписать сообщения.

---

## Пример

```bash
git rebase -i HEAD~5
````

Откроется редактор:

```
pick a1b2c3d Добавил форму логина
pick e4f5g6h Исправил баг в логине
pick i7j8k9l Обновил README
```

Измени `pick` на нужное действие:

|Команда|Что делает|
|---|---|
|`pick`|оставить как есть|
|`reword`|изменить сообщение коммита|
|`squash`|объединить с предыдущим|
|`edit`|остановиться и изменить содержимое|
|`drop`|удалить коммит|

---

## После редактирования

Закрой редактор, Git перепишет историю.  
Если что-то пошло не так:

```bash
git rebase --abort
```
