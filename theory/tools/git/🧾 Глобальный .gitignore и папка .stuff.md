
## Глобальный .gitignore
Чтобы не добавлять IDE-мусор в каждый проект, задай один общий файл:

```bash
git config --global core.excludesfile ~/.gitignore_global
````

`~/.gitignore_global`:

```
.idea/
.vscode/
.stuff/
```

---

## Папка `.stuff`

Теперь ты можешь в любом проекте в любом месте создавать папку `.stuff`:

```
project/
 ├─ src/
 ├─ tests/
 ├─ .stuff/
     ├─ notes.md
     ├─ test.php
```

Она:

- не коммитится (уже в игноре),
    
- подходит для заметок и экспериментов,
    
- может использовать autoload проекта.
    

> Удобнее, чем PhpStorm scratches — всё рядом, IDE видит автолоад, можно быстро запускать эксперименты.
