
По логам гита можно собирать разные метрики, визуализации, строить KPI (но это такое себе)
Можно писать свои скрипты, но есть и распространенные
##  Инструменты

- **git-extras** — десятки расширений: `git summary`, `git effort`, `git release`.
    
- **git-quick-stats** — графики и статистика по авторам/файлам.
    
- **git-fame** — считает долю вклада в проект.
    
- **gitinspector** — аналитика коммитов, полезно для ревью и аудита.
    

Установка:

```bash
brew install git-extras git-quick-stats
```
