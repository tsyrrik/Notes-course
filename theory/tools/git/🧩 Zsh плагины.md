
Если используешь `oh-my-zsh` или `zinit` — включи `git`-плагин.  
Он добавляет кучу коротких алиасов:

|Команда|Расшифровка|
|---|---|
|`gst`|git status|
|`gp`|git push|
|`gl`|git pull|
|`gf`|git fetch|
|`gco`|git checkout|
|`gaa`|git add --all|
|`gcm`|git commit -m|
|`gcam`|git commit -am|
|`gss`|git status -s|

Для статистики и истории можно поставить плагины:

- [`git-extras`](https://github.com/tj/git-extras) — статистика коммитов, активность по файлам;
    
- `git-standup` — кто коммитил за день;
    
- `git-quick-stats` — графики и топ-изменяемые файлы.
