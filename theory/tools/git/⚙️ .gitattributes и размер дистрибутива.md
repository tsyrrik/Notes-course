
## Что делает `.gitattributes`
- задаёт правила diff, merge и export;
- управляет тем, какие файлы попадут в пакет при `git archive` (имеет значение при публикации на Packagist)

---

## Примеры

```gitattributes
*.php diff=php
*.lock -diff
/tests export-ignore
/.github export-ignore
````

Таким образом, тесты, конфиги CI и документация не попадут в релизную сборку.

---

## Оптимизация размера пакетов

Используй [savinmikhail/dist-size-optimizer](https://github.com/savinmikhail/dist-size-optimizer):

```bash
composer global require savinmikhail/dist-size-optimizer
dist-size-optimizer analyze
```

Покажет, какие файлы можно исключить, чтобы не тянуть мусор пользователям библиотеки.
