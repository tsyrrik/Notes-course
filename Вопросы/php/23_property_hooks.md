# Property hooks (PHP 8.4)
Простыми словами: можно объявить `get/set` прямо на свойстве, без явных методов, с типами и валидацией — лаконичная замена шаблонным геттерам/сеттерам.
```php
public int $age {
    get => $this->ageInternal;
    set(int $v) => $this->ageInternal = max(0, $v);
}
```
Даёт лаконичный контроль доступа и валидацию без шаблонных методов.
