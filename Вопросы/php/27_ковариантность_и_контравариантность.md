# Ковариантность и контравариантность
- Ковариантность: возвращаемый тип можно сузить (Dog вместо Animal).
- Контравариантность: аргументы можно обобщить (Animal вместо Dog).
Основано на принципе Лисков (LSP); обратные изменения типов нарушают подстановку.

Пример возвратов (ковариантность)
```php
class Animal {} class Dog extends Animal {}
class Factory { public function create(): Animal { return new Animal(); } }
class DogFactory extends Factory { public function create(): Dog { return new Dog(); } }
```

Пример аргументов (контравариантность)
```php
class Trainer { public function train(Dog $dog) {} }
class MasterTrainer extends Trainer { public function train(Animal $animal) {} }
```
