# Задачи на синтаксис
- Ссылки/COW: `$a=&$b; $b=2; echo $a; //2`
- `unset($a); echo isset($a); // false`
- Статическая переменная в функции: при повторном вызове сохраняет значение.
- Подводные камни сравнения: `"0" == 0 // true`, `'0e123' == '0' // true` (осторожно с хешами!); с `===` эти случаи false.
- Приведение: `(bool)[] // false`, `(bool)"0" // false`, `(bool)"false" // true`.
- Очередность/ассоциативность: `$a = $b = 1;` — сначала `$b=1`, потом `$a=$b`.

## Простыми словами
- Набор напоминаний о тонкостях синтаксиса: ссылки/COW, статические переменные, подводные камни сравнений и приведения типов.
