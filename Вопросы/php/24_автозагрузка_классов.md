## Вопрос: Автозагрузка классов
Версия: PHP 8.4; где есть исторические детали — см. по тексту.

## Простой ответ
не подключаем файлы руками — Composer (PSR-4) или свой автозагрузчик подгружает класс при первом `use/new`.

## Ответ
- Composer: описать PSR-4 в `composer.json`, вызвать `require vendor/autoload.php`.
- Без Composer: `spl_autoload_register()` и маппинг класса → путь.

Пример Composer PSR-4
```json
{
  "autoload": { "psr-4": { "App\\\\": "src/" } }
}
```
```php
require_once __DIR__ . '/vendor/autoload.php';
$service = new App\\Service\\Mailer(); // класс загрузится из src/Service/Mailer.php
```
