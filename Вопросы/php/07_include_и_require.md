## Вопрос: include vs require
Версия: PHP 8.4; где есть исторические детали — см. по тексту.

## Простой ответ
`require` кидает фатал при отсутствии файла (обязательный), `include` — warning и идёт дальше (необязательный). Суффикс `_once` не даёт повторно подключить.

## Ответ
- `require` — фатальная ошибка, если файл не найден (обязательный файл).
- `include` — предупреждение, выполнение продолжается (необязательный файл).
Используйте *_once, чтобы избежать повторного подключения.

Пример
```php
require 'config.php';          // обязателен
include_once 'sidebar.php';    // может отсутствовать, подключаем один раз
echo 'Продолжаем скрипт';
```
