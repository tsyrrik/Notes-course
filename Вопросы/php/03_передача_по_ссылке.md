# Передача по ссылке

Простыми словами: две переменные указывают на один и тот же zval, изменение одной видно в другой. По умолчанию PHP копирует при записи (COW), ссылки ломают эту оптимизацию.

```php
$b = 1;
$a = &$b; // $a и $b указывают на одно zval
$b = 2;
echo $a; // 2
```
Использовать осторожно:
- Не нужно для оптимизации массивов/объектов: PHP сам делает copy-on-write.
- Уместно только когда функция реально должна менять переданный аргумент (например, парсер, который возвращает остаток строки).
