# Рефлексия
Механизм анализа структуры кода во время выполнения.
Классы: `ReflectionClass`, `ReflectionMethod`, `ReflectionProperty`, `ReflectionFunction`.
Применение: DI, ORM, тесты (доступ к приватным частям), фреймворки, аннотации/атрибуты, автозагрузка.

Пример
```php
$ref = new ReflectionClass(DateTime::class);
echo $ref->getShortName();          // DateTime
echo $ref->getMethod('format')->isPublic(); // true
```
