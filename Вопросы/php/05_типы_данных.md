# Типы данных

Простыми словами: скаляры + сложные + специальные; в PHP 8+ появились union/intersection/`never`, отдельные типы `true/false`, `mixed`, `static`.

- Скалярные: `int`, `float`, `string`, `bool`.
- Сложные: `array`, `object`, `callable`, `iterable`.
- Специальные: `null`, `resource`.
- PHP 8+: `mixed`, `never`, `void`, `false`, `true`, union-тип `Type1|Type2`, intersection `A&B`, `static` в возвратах.
- Строгая типизация для аргументов/возвратов включается `declare(strict_types=1);` (иначе PHP пытается приводить).
