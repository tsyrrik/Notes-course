# == и ===
`==` сравнивает с приведением типов, `===` сравнивает тип и значение без приведения. Используйте `===` для безопасных сравнений.
Простыми словами: `==` может дать сюрпризы из-за приведения (`"0"`, `""`, `"0e123"`, `false`, `[]`), поэтому по умолчанию — только `===`.

Пример ловушки с `==`
```php
var_dump(0 == '0');      // true
var_dump(0 == '');       // true (пустая строка приводится к 0)
var_dump('0e123' == '0'); // true (0e... приводится к 0 — риск с хешами)
var_dump(false == []);   // true
```

Безопасное сравнение
```php
var_dump(0 === '0');     // false (разный тип)
var_dump(false === []);  // false
var_dump('abc' === 'abc'); // true
```
