# Память: int vs bool
Простыми словами: в PHP скаляры лежат в `zval`, разница между `int` и `bool` по памяти/скорости несущественна — важнее структура данных (массивы, объекты) и количество элементов.

Что действительно влияет:
- Большие массивы (хеш-таблицы) и объекты — накладные.
- Copy-on-write экономит память до модификации.
- Избегайте лишних копий больших структур и глубоких вложений; используйте генераторы/итераторы там, где возможны потоки.
