# Магические константы PHP

Простыми словами: предопределённые “служебные” идентификаторы, которые PHP подставляет во время исполнения; начинаются и заканчиваются двойным подчёркиванием.

- `__LINE__` — номер строки.
- `__FILE__` — полный путь к файлу.
- `__DIR__` — каталог файла.
- `__FUNCTION__` — имя функции.
- `__METHOD__` — имя метода (`Class::method`).
- `__CLASS__` — имя класса.
- `__TRAIT__` — имя трейта.
- `__NAMESPACE__` — текущее пространство имён.

Пример
```php
echo __FILE__ . ':' . __LINE__;      // /path/to/file.php:12
echo __CLASS__ . '::' . __METHOD__;  // MyClass::doSomething
```
