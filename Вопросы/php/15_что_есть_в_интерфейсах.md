## Вопрос: Что есть в интерфейсах
Версия: PHP 8.4; где есть исторические детали — см. по тексту.

## Простой ответ
интерфейс задаёт контракт; теперь может включать реализацию по умолчанию и статические методы.

## Ответ
- Методы (всегда public), константы.
- PHP 8.1+: статические методы.
- PHP 8.0+: default-методы (реализация по умолчанию).
- PHP 8.4: типизированные константы, свойства без инициализации (RFC/новинка).


Пример default-метода
```php
interface Pingable {
    public function ping(): bool;
    public function name(): string;
    public function status(): string { return $this->name() . ' is ' . ($this->ping() ? 'up' : 'down'); }
}
```
