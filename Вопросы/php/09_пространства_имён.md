# Пространства имён
Простыми словами: неймспейсы группируют код и убирают конфликты имён; по PSR-4 путь к файлу соответствует пространству имён.
```php
namespace App\Controllers;
class UserController {}
use App\Controllers\UserController;
```
Используйте PSR-4 автозагрузку для соответствия неймспейсов структуре файлов.
