# Генераторы и файберы
Генераторы (`yield`, `yield from`) отдают значения по одному без хранения всей коллекции, экономя память.
Итераторы реализуют методы `current/next/key/valid/rewind` для обхода.
Файберы (PHP 8.1) — легковесные сопрограммы: позволяют приостанавливать и возобновлять выполнение с сохранением стека (мощнее генераторов).

Пример генератора
```php
function numbers(): Generator {
    yield 1;
    yield from [2, 3];
}
foreach (numbers() as $n) { echo $n; } // 123
```
