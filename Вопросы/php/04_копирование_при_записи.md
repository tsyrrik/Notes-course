## Вопрос: Copy-on-Write (COW)
Версия: PHP 8.4; где есть исторические детали — см. по тексту.

## Простой ответ
присваиваем массив/объект — копии нет; копирование происходит только в момент изменения одной из ссылок.

## Ответ
```php
$a = [1, 2, 3];
$b = $a;   // копии данных нет
$b[] = 4;  // теперь создаётся копия для $b
```
Экономит память/время при передаче больших структур в функции/методы. Ссылки (`&`) ломают COW и снимают эти оптимизации — используйте их только при реальной необходимости менять аргументы.
