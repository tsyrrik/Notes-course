# Типы БД
- Колончатые (ClickHouse) — аналитика.
- Графовые (Neo4j) — связи.
- Документоориентированные (MongoDB) — JSON-документы.
- Key-Value (Redis).
- Реляционные (PostgreSQL/MySQL) — ACID, SQL.
- Time Series (InfluxDB), NewSQL (CockroachDB/YugabyteDB), Vector DB (Pinecone/Weaviate).

## Простыми словами
- Выбор зависит от модели данных и нагрузки: CRUD/транзакции → реляционные, аналитика → колонка, связи → граф, кэш → key-value, лог/метрики → time-series.
