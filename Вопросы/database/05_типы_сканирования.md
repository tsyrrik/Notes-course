# Типы сканирования
- Sequential Scan — полное сканирование таблицы.
- Index Scan — чтение индекса + обращение к таблице.
- Bitmap Index + Heap Scan — комбинирование нескольких условий.
- Index Only Scan — данные берутся из индекса без чтения таблицы.

## Простыми словами
- Seq Scan — читаем таблицу целиком; ок для малых таблиц или без селективных фильтров.
- Index Scan — точечное чтение через индекс.
- Bitmap — объединяем несколько индексов/условий, читаем строки пачками.
- Index Only — быстрее, если все нужные колонки уже в индексе.

Примеры
```sql
EXPLAIN SELECT * FROM users WHERE email = 'a@b.c';
-- Index Scan using idx_users_email

EXPLAIN SELECT id FROM users WHERE email = 'a@b.c';
-- Index Only Scan, если индекс покрывает колонку id,email

EXPLAIN SELECT * FROM orders WHERE status='paid' AND created_at > now()-interval '7 days';
-- Bitmap Index Scan по двум индексам, затем Bitmap Heap Scan
```
