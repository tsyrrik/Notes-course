# ClickHouse
Колоночная OLAP-СУБД для быстрых агрегатов по большим данным. Поддерживает шардинг/репликацию, сжатие, JSON/массивы, materialized views, TTL. Применение: веб-аналитика, логи, BI, time series.

## Простыми словами
- Очень быстрая аналитика по большим таблицам за счёт колоночного хранения и сжатия.
- Хорош для логов/метрик/отчётов, но не для OLTP и транзакций.
