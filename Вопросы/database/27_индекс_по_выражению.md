# Индексы по выражению
Позволяют индексировать функцию/выражение.
Пример: `CREATE INDEX idx_lower_name ON users (LOWER(name));`
Используется, если запрос фильтрует тем же выражением.

## Простыми словами
- Индексируем не колонку напрямую, а её преобразование.
- Работает только если запрос использует то же самое выражение.
