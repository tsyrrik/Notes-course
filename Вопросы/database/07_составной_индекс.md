# Составной индекс
Индекс по нескольким полям. Используется слева направо: `(a,b)` подходит для `WHERE a=...` и `WHERE a=... AND b=...`, но не для `WHERE b=...`.

Пример
```sql
CREATE INDEX idx_orders_user_status ON orders (user_id, status);
-- WHERE user_id = 5 AND status = 'paid'   -- использует
-- WHERE status = 'paid'                   -- не использует (нет ведущего user_id)
```
