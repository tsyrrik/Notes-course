## Вопрос: Консольные команды
Версия: Yii2 2.0.54; Yii3 релизнут (2025‑12‑31).

## Простой ответ
Для миграций, очередей, кронов и сервисных задач.

## Ответ

Консоль в Yii2 используется для задач без HTTP: миграции, очереди, кроны, импорты, генерация отчётов. Создаются контроллеры в `commands/` (basic) или `console/controllers/` (advanced), их экшены вызываются как `php yii controller/action`.

Пример команды:
```php
class ReportController extends yii\\console\\Controller {
    public function actionDaily() {
        // логика
    }
}
```

Как запускать:
- `php yii report/daily`.
- Поддерживаются `cron` и `queue` интеграции.

## Примеры

1. Команда `php yii migrate` для наката миграций.
2. Крон: `php yii report/daily` каждую ночь.
3. Очереди: `php yii queue/listen` для фоновой обработки.

## Доп. теория

1. Консольное приложение — отдельный конфиг (`console.php`).
2. Для больших задач используйте очереди вместо долгих консольных скриптов.
