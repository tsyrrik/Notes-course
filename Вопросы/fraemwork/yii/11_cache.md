## Вопрос: Кэширование
Версия: Yii2 (ветка 2.0.x); Yii3 в разработке.

## Простой ответ
Кэш — отдельный компонент, можно кэшировать данные, страницы и фрагменты.

## Ответ
## Вопрос: Как устроено кэширование?
Ответ: Кэш — отдельный компонент, можно кэшировать данные, страницы и фрагменты.

## Вопрос: Какие есть драйверы кэша?
Ответ: FileCache, RedisCache, MemCache и др.

## Вопрос: Пример кэширования данных
Ответ:
```php
$data = Yii::$app->cache->getOrSet('key', function () {
    return HeavyService::calc();
}, 60);
```

## Вопрос: Пример фрагментного кэша во view
Ответ:
```php
if ($this->beginCache('list', ['duration' => 60])) {
    echo $this->render('_list');
    $this->endCache();
}
```
