## Вопрос: Фильтры и behaviors
Версия: Yii2 (ветка 2.0.x); Yii3 в разработке.

## Простой ответ
Механизм, который выполняется до или после action (например, проверка доступа или HTTP-методов).

## Ответ
## Вопрос: Что такое фильтры в Yii2?
Ответ: Механизм, который выполняется до или после action (например, проверка доступа или HTTP-методов).

## Вопрос: Какие фильтры используются чаще всего?
Ответ:
- `AccessControl` — доступ по ролям/правам.
- `VerbFilter` — ограничение методов (GET/POST).
- `ContentNegotiator` — форматы ответа.

## Вопрос: Как подключить фильтр?
Ответ:
```php
public function behaviors() {
    return [
        'access' => [
            'class' => yii\\filters\\AccessControl::class,
            'rules' => [
                ['allow' => true, 'roles' => ['@']],
            ],
        ],
    ];
}
```
