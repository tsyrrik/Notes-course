## Вопрос: Events и Behaviors
Версия: Yii2 (ветка 2.0.x); Yii3 в разработке.

## Простой ответ
Механизм подписки на события жизненного цикла (например, перед вставкой в БД).

## Ответ
## Вопрос: Что такое events в Yii2?
Ответ: Механизм подписки на события жизненного цикла (например, перед вставкой в БД).

## Вопрос: Что такое behaviors?
Ответ: Переиспользуемые куски логики, которые можно подключать к моделям/компонентам.

## Вопрос: Пример behavior
Ответ:
```php
public function behaviors() {
    return [
        yii\\behaviors\\TimestampBehavior::class,
    ];
}
```

## Вопрос: Где обычно используются события?
Ответ:
- `EVENT_BEFORE_INSERT`, `EVENT_AFTER_UPDATE`.
- Подписка через `on()` или в `behaviors()`.
