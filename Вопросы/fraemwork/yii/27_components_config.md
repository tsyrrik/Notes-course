## Вопрос: Компоненты приложения
Версия: Yii2 2.0.54; Yii3 релизнут (2025‑12‑31).

## Простой ответ
Сервисы приложения, доступные через `Yii::$app`, настраиваются в конфиге.

## Ответ

Компоненты — сервисы приложения, доступные через `Yii::$app`, настраиваются в конфиге.

Пример определения компонента:
```php
'components' => [
    'payment' => [
        'class' => app\\components\\PaymentService::class,
    ],
],
```

Как использовать:
`Yii::$app->payment->charge($orderId);`

## Примеры

1. Компонент `formatter` для форматирования дат и чисел.
2. Пользовательский компонент `payment` в `components`.
3. Lazy‑loading: компонент создаётся при первом обращении.

## Доп. теория

1. Компоненты наследуют `yii\base\Component` и поддерживают события.
2. Переопределение компонента в local‑конфиге удобно для dev.
