## Вопрос: Компоненты приложения
Версия: Yii2 (ветка 2.0.x); Yii3 в разработке.

## Простой ответ
Сервисы приложения, доступные через `Yii::$app`, настраиваются в конфиге.

## Ответ
## Вопрос: Что такое компоненты Yii2?
Ответ: Сервисы приложения, доступные через `Yii::$app`, настраиваются в конфиге.

## Вопрос: Пример определения компонента
Ответ:
```php
'components' => [
    'payment' => [
        'class' => app\\components\\PaymentService::class,
    ],
],
```

## Вопрос: Как использовать?
Ответ: `Yii::$app->payment->charge($orderId);`
