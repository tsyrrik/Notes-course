## Вопрос: Основные компоненты Symfony
Версия: Symfony 8.0 (stable), LTS 7.4; PHP 8.4+ / 8.2+.

## Простой ответ
Symfony — это конструктор из компонентов; сам фреймворк и другие проекты (Laravel, Drupal) используют их по отдельности.

## Ответ
- HttpFoundation — Request/Response, сессии, куки.
- Routing — маршрутизация URL → контроллер.
- DependencyInjection — DI-контейнер.
- EventDispatcher — события/слушатели.
- Security — аутентификация/авторизация.
- Form — формы, биндинг данных, валидация.
- Validator — правила и проверки.
- Console — CLI команды.
- HttpClient, Messenger, Serializer, Cache, Translation и др.

## Примеры

1. HttpFoundation: `Request::createFromGlobals()`.
2. Console: `php bin/console make:command`.
3. Validator: `#[Assert\Email]` на DTO.

## Доп. теория

1. Компоненты можно использовать отдельно от full‑stack фреймворка.
2. Интеграция компонентов в Symfony обеспечивается Bundle‑ами.
