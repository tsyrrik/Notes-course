# Bundle в Symfony

Простыми словами: bundle — «плагин»/пакет с функциональностью (контроллеры, конфиг, ресурсы), который можно подключать/переиспользовать.

- Содержит: код (контроллеры/сервисы), конфиги, ресурсы (шаблоны, переводы), команды.
- Подключение через `config/bundles.php` (Flex делает это автоматически).
- Свои бандлы обычно не нужны для бизнес-кода в modern Symfony — достаточно `src/`.
