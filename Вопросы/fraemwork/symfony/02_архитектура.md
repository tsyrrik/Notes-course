## Вопрос: Архитектура Symfony
Версия: Symfony 7.x (LTS 6.4), синтаксис и подходы 6.4+.

## Простой ответ
ядро HttpKernel получает запрос, прогоняет через событийнyю систему (middleware), роутит, вызывает контроллер, возвращает Response. Всё строится на сервис-контейнере.

## Ответ
- Request → Kernel → EventDispatcher → Controller → Response.
- Сервисы описаны в контейнере (`services.yaml`), автодобавляются через автоконфигурацию/автовайринг.
