# Валидация

Простыми словами: Валидация проверяет входящие данные по правилам (required/email/min и т.п.).
```php
public function store(Request $request) {
    $validated = $request->validate([
        'name' => 'required|string|max:255',
        'email' => 'required|email|unique:users',
        'password' => 'required|min:8|confirmed',
        'age' => 'required|integer|min:18',
    ]);
    User::create($validated);
}

// Кастомные сообщения
$request->validate(['name' => 'required'], ['name.required' => 'Имя обязательно']);

// Conditional
$request->validate([
    'email' => 'required_if:type,email',
    'phone' => 'required_if:type,phone',
]);
```
