## Вопрос: Query Builder
Версия: Laravel 11 (актуально; базовые принципы применимы к 10+).

## Простой ответ
Query Builder строит SQL методами PHP, чтобы не писать сырой SQL и не забывать экранировать параметры.

## Ответ
ОО-интерфейс для построения SQL без “сырого” SQL. Лёгкий, работает с разными драйверами, экранирует параметры.
```php
use Illuminate\Support\Facades\DB;

$users = DB::table('users')
    ->where('active', 1)
    ->orderBy('created_at', 'desc')
    ->get();
```
Отличия от Eloquent: возвращает массивы/коллекции, не модели; быстрее/проще для сложных SQL.
