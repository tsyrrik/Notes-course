## Вопрос: Dependency Injection в контроллерах
Версия: Laravel 11 (актуально; базовые принципы применимы к 10+).

## Простой ответ
DI — зависимости приходят в контроллер автоматически из контейнера.

## Ответ
Laravel резолвит зависимости из контейнера.
```php
class UserController extends Controller {
    public function store(
        UserRequest $request,
        UserService $userService,
        MailService $mailService
    ) {
        $user = $userService->createUser($request->validated());
        $mailService->sendWelcomeEmail($user);
        return redirect()->route('users.index');
    }

    public function show(User $user, Request $request) {
        return view('users.show', compact('user'));
    }
}
```
