# Контроллеры: типы

Простыми словами: Контроллеры — классы-обработчики запросов (обычные, ресурсные, API, одиночные).
- Обычный: `php artisan make:controller UserController`.
- Resource (CRUD): `--resource`; API: `--api --resource`.
- Invokable (single action): `--invokable`.
```php
class UserController extends Controller {
    public function index() {}
    public function store(Request $request) {}
    public function show(User $user) {}
    public function update(Request $request, User $user) {}
    public function destroy(User $user) {}
}
```
