# Миграции и Schema Builder

Простыми словами: Миграции — версия схемы БД в коде: создаём/меняем таблицы через PHP, чтобы команда и CI применяли одинаковую структуру.
Версионный контроль БД: создают/изменяют структуру программно.
```bash
php artisan make:migration create_users_table
```
```php
class CreateUsersTable extends Migration {
    public function up() {
        Schema::create('users', function (Blueprint $table) {
            $table->id();
            $table->string('name');
            $table->string('email')->unique();
            $table->timestamp('email_verified_at')->nullable();
            $table->string('password');
            $table->boolean('active')->default(true);
            $table->timestamps();
            $table->index('email');
        });
    }
    public function down() { Schema::dropIfExists('users'); }
}
```
