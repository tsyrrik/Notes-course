# Именованные маршруты

Простыми словами: Имя маршрута даёт стабильный идентификатор URL, чтобы менять путь без правок по коду.
Даёт централизованное управление URL.
```php
Route::get('/users/{user}', [UserController::class, 'show'])->name('users.show');
$url = route('users.show', ['user' => 1]);
return redirect()->route('users.show', $user);
// Blade
<a href="{{ route('users.show', $user) }}">Показать</a>
```
