# Архитектура Symfony

Простыми словами: ядро HttpKernel получает запрос, прогоняет через событийнyю систему (middleware), роутит, вызывает контроллер, возвращает Response. Всё строится на сервис-контейнере.

- Request → Kernel → EventDispatcher → Controller → Response.
- Сервисы описаны в контейнере (`services.yaml`), автодобавляются через автоконфигурацию/автовайринг.
