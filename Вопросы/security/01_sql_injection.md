# SQL-инъекция
- Подстановка пользовательского ввода в SQL без экранирования/плейсхолдеров.
- Итог: изменение логики запроса, вытаскивание данных, RCE при высоких привилегиях.
## Защита
- Подготовленные выражения и плейсхолдеры, белые списки идентификаторов.
- Экранирование ненадёжно и не покрывает все случаи.
```php
$stmt = $pdo->prepare('SELECT * FROM users WHERE id = :id');
$stmt->execute(['id' => $id]);
```
