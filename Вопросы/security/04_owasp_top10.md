# OWASP Top 10 (кратко)
Простыми словами: список крупнейших классов уязвимостей веб-приложений по версии OWASP.
1. Инъекции.
2. Некорректная аутентификация.
3. Раскрытие чувствительной информации.
4. XXE (внедрение внешних XML-сущностей).
5. Нарушенный контроль доступа.
6. Security Misconfiguration (ошибки конфигурации).
7. XSS.
8. Небезопасная десериализация.
9. Использование компонентов с известными уязвимостями.
10. Недостаточное логирование и мониторинг.
