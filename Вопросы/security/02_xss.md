# XSS (Cross-Site Scripting)
- Внедрение и исполнение произвольного JS в браузере пользователя.
- Использует авторизацию жертвы для кражи данных/сессий.
Простыми словами: злоумышленник вставляет JS в страницу (комментарий, поле ввода), скрипт ворует куки/делает запросы от имени жертвы.

## Простыми словами
- Не доверяйте пользовательскому вводу при выводе в HTML/JS. Экранируйте контекстно, ставьте HttpOnly/SameSite на куки и CSP.

## Защита
- Экранировать/санитизировать вывод, особенно HTML/JS контекст.
- `HttpOnly` для кук, CSP, валидация входа.
```html
<!-- пример эскейпа -->
<?= htmlspecialchars($userInput, ENT_QUOTES, 'UTF-8') ?>
```
