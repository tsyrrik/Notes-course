## Вопрос: Что такое Kibana

## Простой ответ
Kibana - веб-интерфейс для визуализации и анализа данных в Elasticsearch. Предоставляет дашборды, построение графиков, поиск по индексам и инструменты для APM/безопасности/логов. Через Kibana удобно исследовать данные, строить отчёты и управлять настройками Elastic Stack.

## Ответ
Kibana - веб-интерфейс для визуализации и анализа данных в Elasticsearch. Предоставляет дашборды, построение графиков, поиск по индексам и инструменты для APM/безопасности/логов. Через Kibana удобно исследовать данные, строить отчёты и управлять настройками Elastic Stack.

## Примеры

1. Дашборд ошибок: график 5xx по времени.
2. Discover: поиск `level:error AND service:billing`.

## Доп. теория

1. Kibana работает поверх индексов Elasticsearch и не хранит данные отдельно.
2. Пространства (Spaces) помогают разделять дашборды по командам/проектам.
