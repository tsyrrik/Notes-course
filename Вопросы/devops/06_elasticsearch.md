## Вопрос: Что такое Elasticsearch

## Простой ответ
Elasticsearch - распределённая система полнотекстового поиска и аналитики, построенная на Apache Lucene. Хранит данные в виде документов JSON, поддерживает горизонтальное масштабирование через шардирование и репликацию, обеспечивает быстрый поиск, агрегации и фильтрацию для журналов, метрик и бизнес-данных.

## Ответ
Elasticsearch - распределённая система полнотекстового поиска и аналитики, построенная на Apache Lucene. Хранит данные в виде документов JSON, поддерживает горизонтальное масштабирование через шардирование и репликацию, обеспечивает быстрый поиск, агрегации и фильтрацию для журналов, метрик и бизнес-данных.

## Примеры

1. Индекс `logs-2026.02.03` и поиск по `level:error`.
2. Агрегация по статусам HTTP для аналитики.

## Доп. теория

1. Шарды — единицы горизонтального масштабирования, реплики — отказоустойчивость.
2. Маппинг задаёт типы полей и влияет на скорость/точность поиска.
