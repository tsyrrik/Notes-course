# Интерфейс vs абстрактный класс
Интерфейс задаёт контракт (методы/константы, PHP 8.1+ — статические методы, 8.4 — свойства без инициализации).
Абстрактный класс может содержать реализацию, свойства, конструктор, финальные методы.
Класс может реализовать несколько интерфейсов, но наследует только один абстрактный класс.
