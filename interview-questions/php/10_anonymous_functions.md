# Анонимные функции и замыкания
Функции без имени (объект `Closure`), можно передавать как callback.
```php
$greet = function($name){ echo "Hello, $name"; };
$msg = 'Hi';
$hello = function($name) use ($msg){ echo "$msg, $name"; };
```
Стрелочные функции — синтаксический сахар для кратких замыканий с неявным `use` всех внешних переменных по значению.
