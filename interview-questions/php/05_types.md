# Типы данных
Скалярные: `int`, `float`, `string`, `bool`.
Смешанные: `array`, `object`, `callable`, `iterable`.
Специальные: `null`, `resource`.
PHP 8+: `mixed`, `never`, `void`, `false`, `true`, `union`-типы, `static` в возвратах.
Строгая типизация включается `declare(strict_types=1);` (иначе есть привод типов).
